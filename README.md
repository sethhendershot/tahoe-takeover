# Meal Planning App

A Node.js Express application for collecting meal planning data with secure login using express-session.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. For development with auto-restart:
   ```
   npm run dev
   ```

## Features

- Secure login with sessions
- Basic meal planning routes (placeholder)

## API Endpoints

- GET / - Check login status
- POST /login - Login with username
- POST /logout - Logout
- GET /meals - Get meal planning data (requires login)

## Development

The app runs on port 3000 by default.
